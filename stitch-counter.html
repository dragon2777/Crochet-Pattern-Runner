🧶 Crochet Pattern Runner


Pattern Runner Help

Supported Stitches

  • ch — Chain
  • slst — Slip Stitch
  • sc — Single Crochet
  • hdc — Half Double Crochet
  • dc — Double Crochet
  • tr — Treble Crochet
  • inc — Increase
  • dec — Decrease

Syntax Guide

Basic Row

Row 1: sc 6 (6)

Repeats

(sc 1, inc) x6

Row Targets

(18)

The number in parentheses indicates the expected stitch count.

Around

sc around (18)

This fills the row with single crochet stitches.

Example Pattern

Row 1: sc 6 (6)
Row 2: (inc) x6 (12)
Row 3: (sc 1, inc) x6 (18)
Row 4: sc around (18)
